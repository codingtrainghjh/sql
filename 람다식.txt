함수.메소드
=>메소드이름(매개변수)

인터페이스 변수명 = new 인터페이스{} => 이걸 단순화 => '람다식'

'람다식' : 인터페이스 변수명 = ( 매개변수 ) -> {  실행내역  };

람다식은 함수식의 형태로 진행되는 메소드로서 람다식으로 진행을 하고 싶다면 인터페이스는
하나의 메소드만을 가져야 한다.

@FunthionalInterface : 하나의 메소드만을 가지도록 지정함

람다식에서도 외부의 변수를 사용할 수 있음(다만 외부변수는 final속성을 가지고 있음)

※ 주의점

기존 자바에있는 인터페이스중 하나를 끌어와서 람다식을 쓸 수도 있음(일일이 만들 필요X)
